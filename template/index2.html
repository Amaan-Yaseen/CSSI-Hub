Basic HTML Tags

Tags Description
<html></html> It tells the browser that this is a HTML document.
<head></head> It contains all the data.
<title></title> Specifies the tile of the page.
<body></body> Shows the main content that will be visible on your web page.
<h1></h1> to <h6></h6> This is the heading and the sub-headings.
<p></p> Contains the text.
<div></div> Divides the page into parts.
<strong></strong> Used to bold text.
<a href="#"></a> Defines a hyperlink.
<ol></ol> List things in order.
<ul></ul> Lists things without an order.
<li></li> Contains the list itself.
<br> Adds a line break, and is also a self closing tag.
<style></style> Allows the use of CSS in an HTML page.

How do we setup an HTML page and link files?

- A basic HTML page consists of the following :

  • A HTML tag
  • A HEAD tag
  • A BODY tag


- In order to link files we must use the LINK tag which goes under the HEAD tag.

Frequently asked questions


What does HTML stand for?

HTML stands for Hyper Text Markup Language.

Do all HTML tags have a closing tag?

No, there are some tags that do not have a closing tag like img.

Can you change the color of a text in HTML?

Yes, you would need to use the style tag to do so.
