Frequently asked questions


What is the difference between Null and Undefined?

Null means that the variable doesn't exist and it that the their is no value. While Undefined means that the variable hasn't been declared yet.

What are the differences between == and === ?

=== means that the string or value will be converted or change the item to the same type and do a comparison. While === means that it checks if boths values are true if not it will print out false.
